# ECE200P2
ECE 200 Project 2

Group Members:
Chloe Liu
Atlixqueno Ulizes

This project implements a subset of a MIPS emulator in C.

Work division:
- Chloe Liu:
  - load and stores
  - shifts
  - jump and branch

- Atlixqueno Ulizes:
  - ALU
  - multiplication and division
  - exception handling

The emulator fetches, decodes, and executes instructions, updates the register file, and handles HI/LO values.

We tested the code using the provided test files on BlueHive. We also used MARS to compare register outputs for some test cases.
